Problem Title:
Unique Paths

Problem Description:
You are given an m x n grid. Your task is to find the number of unique paths from the top-left corner (cell (1,1)) to the bottom-right corner (cell (m,n)). At any point, you can only move either right or down.

Input Format:
The first line contains two integers m and n, representing the number of rows and columns in the grid.

Output Format:
Print a single integer, the number of unique paths from the top-left to the bottom-right corner.

Constraints:
1 <= m, n <= 100

Sample Input:
3 7

Sample Output:
28

Explanation:
There are 28 unique paths from the top-left to the bottom-right corner in a 3x7 grid, moving only right or down.

Medium
